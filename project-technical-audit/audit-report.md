# Technical SEO Audit

Website: example.com
Industry: Blog

## Objective
Identify technical SEO issues affecting search engine crawling and indexing.

## Tools Used
- Screaming Frog
- Google Search Console
- PageSpeed Insights

## Issues Found

### Broken Links
14 internal links returning 404 errors.

### Missing Meta Descriptions
23 pages missing meta descriptions.

### Slow Page Speed
Mobile LCP = 4.2 seconds.

## Recommendations

- Fix broken internal links
- Add optimized meta descriptions
- Compress images and defer JavaScript

## Expected SEO Impact

- Improved crawl efficiency
- Better indexing
- Faster load times
